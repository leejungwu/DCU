# DCU
[프로젝트 링크](http://dcutime.shop)

- 웹 풀스택 강의를 듣고 커스텀한 프로젝트
- 교내학생들을 위한 sns
- 교내 sw전시대회 제출

## Front
- Node.js
- Next.js
- React
- Redux saga
- ant design

## back
- Node.js
- MySQL  

## 기능
- 회원가입, 로그인, 로그아웃
- 게시글, 이미지업로드, 해시태그, 수정, 삭제
- 좋아요, 리트윗, 댓글
- 팔로잉
- 해시태그 검색
